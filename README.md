## 📱 What is antigravity-ide-mobile?

antigravity-ide-mobile is an app that lets you control the Antigravity IDE from your mobile device. It gives you access to live chat with AI, file browsing, Git controls, screen mirroring, terminal access, and an admin panel. The interface is built with modern tools like Preact, Vite, Tailwind CSS, and TypeScript.

This tool makes it easier to manage your coding projects remotely without sitting at your computer.
